# Project Reference

This project should track the Gabriel e Oscar Wink Studio experience shown in the reference screenshot saved at:

`reports/project-reference-wink-studio-2026-06-26.png`

Reference URL shown in the browser: `https://novaswinks.vercel.app`

Key notes:

- Dark navy/purple app shell with a left sidebar.
- Brand: `GABRIEL E OSCAR` / `WINK STUDIO`.
- Main sections: `Chat Winks`, `Full Bingo Winks`, `Sound`, `Favorites`, `Downloads`, `Categories`, `Settings`, `Help`.
- Primary content shows wink cards with transparent preview area, purple play button, favorite button, APNG badge, resolution badge, download button, and preview button.
- Chat Winks mode uses `APNG 768x1024`.
- Full Bingo Winks mode uses `LOTTIE / JSON 1920x1024`.
- Overall visual target: premium neon bingo/wink studio, dark interface, purple accents, compact card grid.
